# News page

Simple news page.

# Technologies used

- Nuxt3
- Taiwlind
- Pinia
- Typescript
- apexcharts

## Installation

1. Build and run dev.

```bash
npm run install
npm run dev
```
2. Copy env.example to .env and insert weather api keys and news api keys.

```bash
cp .env.example .env
```
